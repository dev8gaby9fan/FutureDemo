# FutureDemo

一款开源的期货行情、交易 Android app

## 功能

1. 支持期货合约加载和显示

支持中金所、大商所、上期所、郑商所和能源所期货合约。合约和行情均来自于快期开源项目[快期开源项目](https://github.com/shinnytech/shinny-futures-android)

2. 支持CTP直连柜台交易

使用CPT API接口，支持看穿式监管

3. 支持Sim now模拟交易

内置了盘中和盘后两套环境，同时应用也支持新增柜台
